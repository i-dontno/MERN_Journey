Manjero linux installation Guide by AJ

Welcome to this my first blog. Here we are going to see how we install the Manjero linux distro. This Linux distro is based on Arch linux

Prerequisite

Before we start the instalation we should have following

  • Manjero Linux ISO. If not you can download from official site Manero Official site
  • Usb flash drive at least of 2GB or more

Cretae a bootable USB

U can use Rufus or Etcher or Power ISO

Installation steps

  • Insert the bootable USB drive and boot your computer from it.
  • Follow the on-screen instructions to start the installation process.
  • Choose your language, time zone, and keyboard layout.
  • Partition your disk or choose the automatic partitioning option.
  • Set your username and password.
  • Complete the installation and reboot your system
